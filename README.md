# SeatIQ - 新能源汽车座椅备件智能定价与分析平台

## 项目简介

SeatIQ 是一个专业的B端Web平台，为汽车后市场企业的座椅备件定价分析师、采购专员和产品经理提供精准的市场感知、深度研究工具和定价决策支持。

## 核心功能

### 1. 工作台 (Dashboard)
- 实时数据统计
- 价格趋势分析
- 市场份额分布
- 最新价格动态
- 预警通知

### 2. 座椅配置库 (Seat Library)
- 多维度筛选（品牌、车系、年款、材质、功能等）
- 座椅位置层级选择（主副驾、后排、第三排）
- 价格区间筛选
- 保存常用筛选方案
- 产品详情查看
- 加入对比和收藏

### 3. 物流计算器 (Logistics Calculator)
- 拆分件库管理
- 抛重自动计算
- 多物流商报价对比（德邦、顺丰、京东）
- 物流成本占比分析
- 最优方案推荐

### 4. 竞品分析 (Competitor Analysis)
- 综合评分模型（可配置权重）
- 成本效用比模型（可配置系数）
- 预设分析方案（均衡型、功能优先型、成本敏感型、品质优先型）
- 雷达图可视化
- 价格-功能分布图

### 5. 收藏夹 (Favorites)
- 收藏座椅配置
- 快速查看详情
- 批量对比

### 6. 对比列表 (Compare)
- 多产品对比（最多5个）
- 差异高亮显示
- 导出对比报告（PDF/Excel）
- 打印功能

### 7. 分析记录 (History)
- 操作历史记录
- 一键重现
- 结果导出
- 记录搜索和筛选

## 技术栈

- **前端框架**: React 18
- **UI组件库**: Ant Design 5
- **路由**: React Router 6
- **图表库**: ECharts + echarts-for-react
- **构建工具**: Vite
- **日期处理**: Day.js

## 安装和运行

### 前置要求

- Node.js 16+
- npm 或 yarn

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:3000 查看应用

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 功能演示

### 座椅配置库
- 支持按品牌、车系、年款、座椅位置筛选
- 支持按材质类型（Nappa真皮、Alcantara等）筛选
- 支持按功能配置（电动调节、加热、通风、按摩等）筛选
- 支持价格区间筛选
- 可保存常用筛选方案
- 查看产品详情、加入对比和收藏

### 物流计算器
- 选择拆分件自动填充尺寸和重量
- 手动调整包装尺寸和实重
- 自动计算抛重和计费重量
- 对比多个物流商报价
- 显示物流成本占比
- 推荐最优方案

### 竞品分析
- 综合评分模型：价格、功能、材质、权重可配置
- 成本效用比模型：α、β、γ系数可配置
- 预设方案快速切换
- 雷达图展示综合能力
- 散点图展示价格-功能分布

## 数据说明

当前版本使用模拟数据，包含：
- 12款主流新能源车型座椅配置（特斯拉、比亚迪、蔚来、理想、小鹏、极氪）
- 12种拆分件数据
- 5个竞品数据
- 3个物流商费率
